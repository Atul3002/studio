# **App Name**: BreakerTrack

## Core Features:

- Admin Page: Admin Page: Password-protected page for administrators to manage user access and settings.
- Production Team Page: Production Team Page: Password-protected page for the production team to view reports.
- Operator Page: Operator Page: Password-protected page for operators to input production data.
- Machine Selection: Machine Selection: Operators choose the machine they are using from a pre-configured list.
- Daily Production Input: Data Input: Allow operators to log the number of breakers produced per shift.
- Automated Reports: Report Generation: Automatically create reports to help monitor breaker output, based on manual input by the operators. Reports cover daily production.

## Style Guidelines:

- Primary color: Dark slate blue (#37718e) for a professional and serious tone, given the nature of tracking production data.
- Background color: Very light gray (#f0f2f5), a slightly desaturated shade of the primary hue to create a soft background that won't distract from the data.
- Accent color: Muted lavender (#c0b6ce) to draw attention to key interactive elements.
- Body font: 'Inter' sans-serif for clear data presentation and readability.
- Headline font: 'Space Grotesk' sans-serif for titles and headings.
- Use clear and simple icons to represent different production metrics and navigation elements.
- Dashboard layout with clear sections for data input, machine selection, and report viewing.
- Subtle transitions and animations for a smooth user experience.
